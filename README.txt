longitudinal_register README
==================

Getting Started
---------------

- $easy_install pip

- $pip install virtualenv

- $pip install virtualenvwrapper (optional)

- $virtualenv longitudinal_register

- $cd longitudinal_register

- $git clone git@github.com:etoko/longitudinal_register.git

- cd <directory containing this file>

- $pip install -r requirements.txt

- $venv/bin/python setup.py develop

- $venv/bin/initialize_longitudinal_register_db development.ini

- $venv/bin/pserve development.ini
